# svgcode
A minimal SVG to GCODE converter for 2d operations

## Installation

  `npm install svgcode`

## Usage

    const svgcode = require("svgcode");

    const gcode = svgcode()
    .loadFile(__dirname + "test/test.svg")
    .generateGcode()
    .getGcode();
    
    console.log(gcode)
  


## Tests

  `npm test`
